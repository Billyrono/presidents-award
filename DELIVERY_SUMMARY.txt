QUICK START INSTRUCTIONS
================================================================================

1. INSTALL DEPENDENCIES
   Command: npm install
   Time: 2-3 minutes

2. START DEVELOPMENT SERVER
   Command: npm run dev
   Time: Instant
   URL: http://localhost:3000

3. VIEW IN BROWSER
   The website will open with all 8 sections
   Test navigation, form, and responsiveness

4. BUILD FOR PRODUCTION
   Command: npm run build
   Creates optimized production build

5. DEPLOY
   Option 1: Vercel (recommended)
   - Push to GitHub
   - Connect to Vercel
   - Auto-deploy on push

   Option 2: Netlify
   - Push to GitHub
   - Connect to Netlify
   - Auto-deploy on push

   Option 3: Manual
   - Run: npm run build
   - Upload /out folder to server
